# Student Management Backend

 Features
- Create, Read, Update, Delete Students
- DTO Validation using Jakarta Validation
- Global Exception Handling
- RESTful APIs

 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

 API Endpoints
- POST /student
- GET /students
- GET /students/{id}
- PUT /students/{id}
- DELETE /students/{id}
